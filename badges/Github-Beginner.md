# 🚀 GitHub for Beginners Journey

This repository documents my progress in mastering the fundamentals of Git and GitHub. It follows the structured learning path from the [Github-for-beginners](https://github.com/nisalgunawardhana/Github-for-beginners) curriculum.


---

## 📚 What You'll Learn
Through this repository, I am covering the fundamental skills of the GitHub ecosystem:
* **Forking & Cloning:** Learning to fork repositories and clone them to my local machine.
* **Branching Strategy:** Creating, switching, and managing branches.
* **Workflow:** Making changes to files, staging, committing, and pushing to GitHub.
* **Collaboration:** Creating Pull Requests and managing Issues.
* **Git Essentials:** Mastering basic Git commands and the standard GitHub workflow.
